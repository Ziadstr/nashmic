  NashmiC — نشمي
  ═══════════════
  A programming language from Jordan.
  Easy like Python. Powerful like Go.
  Jordanian to the bone.

  نشمي (Nashmi) = brave, noble, honorable.
  The highest compliment in Jordanian culture.

  Features:
  • Franco-Arab keywords — code the way you text
  • natije<T> + wala? — error handling with personality
  • yimkin<T> — no null, ever
  • String interpolation — "marhaba ya {name}"
  • Compiler proverbs — learn wisdom with your errors
  • drobi() — when you need a laugh

  Built by Ziad Al Sharif 🇯🇴
